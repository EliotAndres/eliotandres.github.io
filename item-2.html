So You Like That Song

Music recommendation (Demo)

The project

I was frustrated with Soundcloud's recommendations (it got a lot better lately !). I decided to build my own. You select a track (the seed) and the app fetches playlists containing it. It then stacks all the tracks and sorts them by number of occurences.

The stack

The front-end uses Angular.js and the backend is is plain node.js.
