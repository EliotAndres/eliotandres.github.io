Pop Valet

Uber for parking // Made at BAM

The project

Getting to a great user experience takes time: a seven-week project like Pop Valet gave us plenty of time to try, fail and iterate. The main purpose of the app is to give you access to premium valets : trusting the service was extremely important.

The stack

Although I did't mess a lot with it, sails was used for the backend. Administration is done using ng-admin. For the frontend, along with the classic Ionic + Angular, we also used a native map plugin to avoid reloading maps.
